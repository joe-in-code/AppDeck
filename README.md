# AppDeck

**A flexible workspace for organizing cards, notes, lists, tables, and more. Featuring AI collaboration on multiple files at once!**

## Overview

AppDeck is a versatile card-based workspace that lets you organize information your way. Create notes, lists, tables, task boards, key-value pairs, and other card types in a customizable grid layout.

## Features

- **Multiple Card Types**: Notes, Markdown, Lists, Tables, Tasks, Key-Value pairs, Library browser
- **Flexible Layout**: Drag cards to reorder, resize, minimize/maximize, or free-float, even drag panel cross bars to size!
- **AI Integration**: Chat and Create modes for AI-assisted content generation
- **Import/Export**: Save workspaces as .wsz files, import CSV/MD/JSON into cards
- **Voice Input**: Alt+V for voice commands
- **Search**: Global search across all cards
- **Undo/Redo**: Full history with Ctrl+Z/Y

## Quick Start

1. Click toolbar buttons to add new cards
2. Drag card headers to rearrange
3. Right-click cards for options (save, import, snap, delete)
4. Double-click headers to expand cards to fill panel
5. Use File menu to create new tabs or save workspaces

## Card Types

- **NOTE**: Rich text editor with formatting
- **MD**: Markdown editor and preview
- **LIST**: Hierarchical checklists with indentation
- **TABLE**: Spreadsheet-like data tables
- **TASK**: Kanban-style task board with priority, status, assignments
- **KV**: Key-value pair storage
- **LIBRARY**: Browse and load card templates

## Keyboard Shortcuts

- `Ctrl+Z` / `Ctrl+Y` - Undo/Redo
- `Ctrl+C` - Copy card content
- `Ctrl+E` - Save card as file
- `Ctrl+I` - Import into card
- `Alt+V` - Voice input

## AI Modes

- **Chat Mode**: Conversational responses with confirmation buttons for proposed changes
- **Create Mode**: AI automatically creates and updates cards based on your requests
